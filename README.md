## Versões
* ruby: 2.7.7
* rails: 6.1
* postgresql: 14-3.2
* bootstrap: 5


## Instalando Sistema

1. Primeiro execute

    ```
    bundle install
    ```
    ```
    yarn install
    ```
   
2. Agora execute o projeto

    ```
    foreman start
    ```

3. Crie o banco de dados, migração e seja feliz!

    ```
    rails db:create
    ```
    ```
    rails db:migrate
    ```
